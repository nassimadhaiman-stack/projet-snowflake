# 🧊 LinkedIn Jobs Analysis with Snowflake

## 🎯 Objective

Analyze LinkedIn job postings data to extract insights about job market trends using Snowflake and Streamlit.

## ⚙️ Tools

* Snowflake (SQL)
* Streamlit
* AWS S3 (data source)

## 🧱 Project Steps

### 1. Data Setup

* Created database `linkedin`
* Configured external stage from S3 bucket

### 2. Data Loading

* Defined file formats (CSV, JSON)
* Loaded data using COPY INTO

### 3. Data Transformation

* Cleaned and structured data
* Ensured consistency between tables

### 4. Analysis

* Top job titles per industry
* Highest paying jobs
* Job distribution by company size
* Job distribution by industry
* Job distribution by employment type

### 5. Visualization

* Built dashboards using Streamlit in Snowflake

## 📊 Key Insights

* Some industries have higher salary ranges
* Full-time roles dominate job postings
* Large companies post more jobs

## ⚠️ Challenges

* Handling JSON data
* Data consistency between tables

## 🧠 Conclusion

This project demonstrates my ability to manage data pipelines, perform analysis, and communicate insights effectively.
